# Vendor Reliability Intelligence & Procurement Risk Management Platform

## Project Overview

The Vendor Reliability Intelligence Platform is a full-stack web application designed to help organizations evaluate vendor reliability, manage procurement operations, monitor supplier performance, track delivery history, maintain contract compliance, and improve procurement decision-making through centralized dashboards and analytics.
